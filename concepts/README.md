# ICT Concepts

## 문서 규칙

- 이 폴더는 개념의 의미, 차트 관찰 방법, 다른 개념과의 연결을 설명한다.
- 구현 판정은 대응하는 `../rulebook/*_rulebook.md`를 기준으로 한다.
- `실제 차트 예시`는 자막 영상의 가격 좌표를 추측하지 않는다. OHLC에서 재현할 수 있는 조건과 확인할 자막 Episode를 함께 기록한다.
- 새로운 근거를 추가할 때는 기존 설명을 삭제하기보다 `근거`, `반례`, `미확정` 항목을 확장한다.

## 상태 표기

- `objective`: OHLC 또는 외부 구조화 데이터로 계산 가능
- `research default`: 자막에 고정 수치가 없어서 백테스트용 초기값을 둔 것
- `manual_review`: 영상 차트 문맥 또는 재량 판단이 남아 있는 것
